# Welcome to the Knowledge Base

This public website consolidates authoritative content from multiple sources into one searchable hub:

- **IDSA Rulebook** → governance rules, policies, and processes  
- **IDSA RAM 5** → reference architecture model for data spaces  
- **Organizational Handbook** → operational guidance for members

The site is generated with **MkDocs Material**, and the left navigation is **driven by a merged `SUMMARY.md`** assembled at build time from the upstream repositories’ own summaries. Client‑side search is enabled.

> **Tabs:** Use the header tabs to switch between **Home / Knowledge / About**.
